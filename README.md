# prompt-spark-render-lab

> AI image generator UI (frontend only).

![Status: Experiment](https://img.shields.io/badge/Status-Experiment-yellow)

## What It Does

A clean UI for generating AI images from text prompts. No backend implementation - just the interface for what an image generation tool would look like. Displays current prompt and a grid for generated images.

## Tech Stack

`React` `Vite` `TypeScript` `shadcn/ui` `Tailwind CSS` `TanStack Query`
